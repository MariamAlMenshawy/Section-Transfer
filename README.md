# Section Transfer Website 

A web-based Section Transfer Management System built using the Django framework (Python), HTML, CSS, and Bootstrap.
This project allows students to submit, browse, and manage section transfer requests in an organized and user-friendly way.


## Features

### Authentication System
* **Secure Login/Logout** 
* **Each student has a personal account** 


### Transfer Request System
* **Students can create transfer requests between sections** 
* **Easy request submission and management** 
* **Browse & Filter System:** View all available transfer requests.
* **Advanced filtering options:** Filter by Academic Year, Filter by, Current Section, Filter by Target Section.
* **Request Management:**Users can delete their own transfer requests using their University ID.
* **Request Details Page:** View full details of each request and Display student contact details.


## Technologies Used 
* ***Backend:*** Python & Django Framework.
* ***Frontend:*** HTML5, CSS3, and Bootstrap for responsive design.
* ***Database:*** PostgreSQL.



## How to Run
1. Clone the repository to your local machine.
2. Ensure Python and Django are installed.
3. Install dependencies `pip install -r requirements.txt` .
4. Run `py manage.py migrate` to set up the database.
5. Start the server using `py manage.py runserver` .
6. Open http://127.0.0.1:8000/ in your browser.
